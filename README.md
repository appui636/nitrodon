# 🎵 IntroDone! ONLINE

曲のイントロを聴いて最初に答えを当てるWebゲーム。URLを知っている人なら誰でも遊べます。

![IntroDone!](https://img.shields.io/badge/status-ready-brightgreen) ![License](https://img.shields.io/badge/license-MIT-blue)

## 🎮 ゲーム概要

- **ホストが部屋を作成** → **6文字コードを友達に共有** → **最大10人で同時対戦**
- 曲のイントロが流れる → **早押しボタン/スペースキー** → 10秒以内に曲名を入力
- 正解 → サビが流れてスコア+1
- 不正解 → その問題は一回休み

### 主な特徴
✅ **リアルタイムマルチプレイ** - WebSocketでサーバーと同期  
✅ **Cookie永続ニックネーム** - 何度でもすぐ遊べる  
✅ **ホスト設定** - 問題数・追加秒数・カテゴリをカスタマイズ  
✅ **検索結果非表示** - robots.noindex で Google検索に引っかからない  
✅ **外部依存ゼロ** - Node.js 標準モジュールのみ  
✅ **自動再生対応** - モダンブラウザの autoplay 制限に自動対応

## 🎵 曲を追加する方法

### 方法1: 管理画面（簡単）

1. ブラウザで `/admin.html` を開く
   ```
   http://localhost:3000/admin.html
   ```

2. MP3ファイルをアップロード & 情報を入力
   - 曲タイトル（正解文字列）
   - アーティスト名
   - カテゴリ（J-POP, アニメ, 洋楽など）
   - ファイル名
   - イントロ再生開始時間（秒）
   - サビの開始時間（秒）
   - サビの再生時間（秒）

3. "追加する" ボタンで自動保存

### 方法2: catalog.json 直接編集

`music/catalog.json` を編集:

```json
{
  "songs": [
    {
      "id": "unique_id_1",
      "title": "ドライフラワー",
      "artist": "優里",
      "category": "J-POP",
      "file": "drymflower.mp3",
      "startTime": 0,
      "sabiBeat": 60,
      "sabiDuration": 15
    }
  ]
}
```

**パラメータ説明:**
- `id`: 一意のID（何でもOK、重複しないように）
- `title`: 正解として認識される曲名（正規化される：スペース・句読点は無視）
- `artist`: アーティスト名（表示用）
- `category`: カテゴリ（ゲーム設定でフィルタリング可能）
- `file`: `music/` フォルダ内のファイル名
- `startTime`: イントロ再生の開始時間（秒、デフォルト0）
- `sabiBeat`: サビの開始時間（秒）
- `sabiDuration`: サビの再生時間（秒、デフォルト15）

### 答え合わせのルール

以下は無視されます（どちらでも正解）:
- 全角/半角スペース
- 中点（・）、長音符（ー）
- 句読点（。、！？）
- 大文字/小文字

例: 以下は全て同一扱い
- `ドライフラワー`
- `ドライフラワ ー`
- `DRY FLOWER`

## ⚙️ ゲームルール

### 部屋の作成者（ホスト）が設定できる項目

| 項目 | 説明 | デフォルト |
|------|------|----------|
| 問題数 | 1ゲームの出題数 | 10問 |
| 追加秒数 | 曲ごとの基本秒数に加算 | 0秒 |
| カテゴリ | 「全て」または特定カテゴリのみ | 全て |

### ゲームの流れ

1. **部屋待機画面** - 参加者が集まるまで待つ
2. **カウントダウン** - 3秒で準備
3. **イントロ再生** - 曲のイントロが流れる
4. **早押し** - 最初に「わかった！」を押した人が回答権獲得
5. **回答入力** - 10秒以内に曲名を入力
6. **正誤判定**
   - ✅ **正解** → サビ再生 → スコア+1 → 次の問題へ
   - ❌ **不正解** → その問題は一回休み → 他の人が続行
7. **全員ミス** → タイムアップ → 次の問題へ
8. **ゲーム終了** - 全問終了後に結果表示

## 🔒 アクセス制限（検索結果非表示）

このサイトは以下で保護されています：

1. **robots.noindex** - Google検索に引っかからない

- Nginx でBasic認証を追加
- IP制限を設定

## 🛠️ トラブルシューティング

### 曲が再生されない

**確認事項:**
```bash
# 1. ファイルが存在するか
ls music/
# → MP3ファイルが表示されるか確認

# 2. catalog.json のファイル名が正しいか
cat music/catalog.json

# 3. パーミッション確認
chmod 644 music/*.mp3
```

**ブラウザのコンソールを確認:**
- F12 キーで開発者ツールを開く
- コンソールタブでエラーを確認
- 404 エラーならファイル名不一致
